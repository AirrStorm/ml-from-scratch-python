# ml-from-scratch-python

A collection of machine learning algorithms implemented **from scratch** in **pure Python** with **no external libraries** (except matplotlib for visualization).

This project focuses on the core logic behind ML algorithms, starting from the ground up to help build a deep understanding of their inner workings.
